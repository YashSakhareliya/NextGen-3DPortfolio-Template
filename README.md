### 1. Clone the repository:
```bash
git clone https://github.com/yashsakhareliya/NextGen-3DPortfolio-Template.git
```

### 2. Install dependencies:
```bash
cd NextGen-3DPortfolio-Template

```

### 3. Start the development server:
```bash

```
